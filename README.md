# Telegram Bot

Простой телеграм бот с базовым функционалом.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

1. Запустите бота:
```bash
python bot.py
```

## Функциональность

- Команда `/start` - запускает бота и показывает меню с кнопками
- Кнопки:
  - "Получить карту"
  - "Моя коллекция"
  - "Меню"
